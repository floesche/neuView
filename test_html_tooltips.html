Responsive Width Demo

This tooltip demonstrates how the width adapts to different screen sizes:

  • Mobile (≤ 768px): Max 95vw width, smaller font/padding
  • Desktop (1200px+): Up to 500px width
  • Large screens (1600px+): Up to 600px width
  • Very small (≤ 480px): Forces center positioning for better readability
